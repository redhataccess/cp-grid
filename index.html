Standard Grid

You can assign the column/grid widths directly to the grid wrapper itself using the data attribute data-cp-grid. Breakpoints work similarly to Bootstrap.

Markup:

<div class="cp-grid" data-cp-grid="sm-2 md-4">
		<div class="cp-grid-item"></div>
		<div class="cp-grid-item"></div>
		<div class="cp-grid-item"></div>
		<div class="cp-grid-item"></div>
	</div>

Grid with Cards

The truth is, you don't need cards anymore. The grid can have backgrounds now. By adding the data attribute data-cp-grid-bg you can select from any of the Brand-approved card colors:

  • none — this will add the necessary padding, but with a transparent background.
  • light-grey
  • medium-grey
  • dark-grey
  • bordered

Markup:

<div class="cp-grid" data-cp-grid="sm-2 md-4">
	<div class="cp-grid-item" data-cp-grid-bg="light-grey"></div>
	<div class="cp-grid-item" data-cp-grid-bg="light-grey"></div>
	<div class="cp-grid-item" data-cp-grid-bg="none"></div>
	<div class="cp-grid-item" data-cp-grid-bg="bordered"></div>
</div>

Grid with Varying Widths

By adding the data attribute data-cp-grid-size you can adjust individual cp-grid-item widths. The values work similarly to Bootstrap as well.

Note: These will only override their respective grid wrapper size. Therefore, if you have a value of md-4 on the wrapper, you will need a value on the cp-grid-item to override this.

Markup:

<div class="cp-grid" data-cp-grid="sm-2 md-4">
	<div class="cp-grid-item" data-cp-grid-bg="light-grey" data-cp-grid-size="sm-12-cols md-6-cols"></div>
	<div class="cp-grid-item" data-cp-grid-bg="light-grey"></div>
	<div class="cp-grid-item" data-cp-grid-bg="light-grey"></div>
	<div class="cp-grid-item" data-cp-grid-bg="light-grey"></div>
</div>

Grid with Alignment

By adding the data attribute data-cp-grid-align to the wrapper you can vertically align all the elements. Or, you may choose to align only an individual element.

Note: EqualHeights (or "stretching") cannot be used alongside vertical alignment. This is the nature of Flexbox.

Markup:

<div class="cp-grid" data-cp-grid="sm-2 md-4" data-cp-grid-align="middle">
	<div class="cp-grid-item"></div>
	<div class="cp-grid-item"></div>
	<div class="cp-grid-item"></div>
	<div class="cp-grid-item"></div>
</div>
