Standard Grid

You can assign the column/grid widths directly to the grid wrapper itself using the data attribute data-cp-grid. Breakpoints work similarly to Bootstrap.

Markup:

<div class="cp-grid" data-cp-grid="sm-2 md-4">
	<div class="cp-grid-item"></div>
	<div class="cp-grid-item"></div>
	<div class="cp-grid-item"></div>
	<div class="cp-grid-item"></div>
</div>
